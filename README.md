# Ouvidor
Bot que escuta a radio educadora e tenta salvar tudo que eh dito na rádio.

#Instalação
git clone https://github.com/mthbernardes/Ouvidor.git
pip install -r dependencies.txt

#Mac OS
<pre>
brew install rtmpdump
pip install -r dependencies.txt
</pre>
